# CloudThinker Connections

This repository stores connection integrations and binaries for the CloudThinker platform.

## Structure

- `binaries/` — Compiled CLI tools and executables used by CloudThinker connections
  - `ct-zabbix-cli` — CLI tool for interfacing with Zabbix monitoring

## About

CloudThinker Connections provides the tooling and binaries needed to integrate CloudThinker with external services and monitoring platforms.
